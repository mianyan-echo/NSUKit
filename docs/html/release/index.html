NSUKit为一套用于各种射频/数字组件的SDK。支持c++、python等编程语言，此接口完成了 主机对板级交互的层级抽象 ，目标是 分割上层应用程序和多种多样的板级IO接口 ，可以实现 一次开发、多Soc兼容 。

本文档为NSUKit的C++语言版本的使用文档。
