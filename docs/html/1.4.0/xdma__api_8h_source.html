164//Start a sending (h2c) dma.
165//boardn: board index.
166//chnl: dma channel. 0..3 (depends on IP)
167//dma: the dma object carries the data buffer.
168//len32: transfer length in 32bit words.
169//offset: data location (offset) in the dma buffer.
170//last: if it has a AXIS TLAST at the end of dma. Only for streaming.
171//mm_addr: address on the AXI4 MM interface. Only for MM.
172//mm_addr_inc: If the AXI4 MM address increases.
173//timeout: dma waiting timeout in milliseconds. -1= infinite. 0=no wait.
174//return the transfored data length in DWORDs.
175//return DMA_START_ERROR for dma start error. use fpga_err_msg() to the get detial error information.
