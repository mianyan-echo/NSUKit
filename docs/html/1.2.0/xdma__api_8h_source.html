148//Start a receiving (c2h) dma
149//boardn: pciBoard index.
150//chnl: dma channel. 0..3 (depends on IP)
151//dma: the dma object carries the data buffer.
152//len32: transfer length in 32bit words.
153//offset: data location (offset) in the dma buffer.
154//last: if it has a AXIS TLAST at the end of dma. Only for streaming.
155//mm_addr: address on the AXI4 MM interface. Only for MM.
156//mm_addr_inc: If the AXI4 MM address increases.
157//timeout: dma waiting timeout in milliseconds. DMA_WAIT_FOR_EVER= infinite. 0=no wait.
158//return the transfored data length in DWORDs.
159//return DMA_START_ERROR for dma start error. use fpga_err_msg() to the get detial error information.
160DLLEXPORT unsigned long long _API_CALL
